# Sistema RPA para la automatización de procesos administrativos

## Descripción

Este repositorio contiene el código fuente desarrollado como parte del Trabajo Fin de Grado **"Automatización de procesos administrativos mediante Robotic Process Automation (RPA) en un entorno empresarial real"**.

El sistema ha sido desarrollado utilizando AutoIt y automatiza el proceso de gestión de incidencias entre una plataforma de gestión de incidencias y un sistema interno de registro, reduciendo la intervención manual y mejorando la eficiencia del proceso.

---

## Funcionalidades principales

- Detección automática de incidencias pendientes.
- Validación visual mediante reconocimiento de estados.
- Creación automática de registros.
- Gestión de errores y recuperación del proceso.
- Generación automática de logs de ejecución.
- Registro del tiempo de ejecución y trazabilidad del sistema.

---

## Tecnologías utilizadas

- AutoIt
- Microsoft Windows
- Microsoft Edge
- GitHub

---

## Nota

Por motivos de confidencialidad, se han eliminado del código todas las credenciales, datos personales e información interna de la empresa.

Este repositorio se publica únicamente con fines académicos como material complementario del Trabajo Fin de Grado.
